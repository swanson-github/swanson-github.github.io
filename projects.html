C# - academic and work project

Based on taking Programming Concepts at NTC and my current work with control systems, I recently completed a C# console application. This application can be configured to read JSON from the command line as a string or from a file. It then processes the specifically formatted JSON to write the data to a PLC for use in controlling industrial machinery.

 

Node-Red - work project

I am currently working with Node-Red as a low code middleware for a project to communicate between a User Interface and a PLC backend processor. The UI sits on one PC and is the main point of data entry and feedback for the operator to run a piece of industrial machinery used in the paper converting industry. The Node-Red sits on a separate PC and communicates the user instructions to the PLC.
